DragonSoul

UI Implementation

My work on DragonSoul primarily consisted of tweaking UI elements using responsive design principles and 9-slicing. Specific work includes:

  • Creating a list of known Android screen sizes, dynamically adjusting screen size based on device information.

  • Scaling buttons and re-positioning buttons to keep layout consistent among different screen sizes

  • Re-positioning and centering treasure chest microtransaction elements
